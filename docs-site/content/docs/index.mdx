Bem-vindo à documentação do **Clinica** — o sistema de gestão para clínicas de
psicologia (e outras áreas da saúde). Aqui você encontra o passo a passo para
configurar a clínica, cadastrar a equipe e os pacientes, usar a agenda, o
prontuário, o financeiro, o fiscal e o portal do paciente.

Use o **menu lateral** ou a **busca** (atalho no topo) para navegar. Se é a sua
primeira vez, comece por **Primeiros passos** e depois **Configuração inicial**.
